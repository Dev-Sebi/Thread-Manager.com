Features of Thread Manager
Thread Manager is a bot that will look over all threads created in your Server and logs all activities made on or in them, for example:
Deleted and Updated Messages

Thread Manager will also look out for Messages inside of Threads to be Updated or Deleted and will also log them into the by you selected logging channel

Created Threads

Thread Manager will look out for new Threads to be created and logs them into your selected logging Channel.

Deleted Threads

Thread Manager will look out for Threads to be deleted and logs them into your selected logging Channel.

Updated Threads

Thread Manager will look out for Threads to be updated for example name changings or other and logs them into your selected logging Channel.
